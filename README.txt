Student Information Portal
==========================

This is a web-based portal for managing student information and identification at S.S.S.S.S. PU. GOVT. POLYTECHNIC SULT ALMORA.

Features:
---------
1. Add new student records with details like:
   - Name
   - Roll Number
   - Contact Number
   - Email ID
   - Age
   - Registration Number
   - Gender
   - Year (1st, 2nd, 3rd)
   - Branch (CSE or Elex)
   - Standard Photo

2. View all student records in a searchable table

3. Filter students by year and branch

4. Edit or delete student records

5. Statistics showing total students and distribution across years

How to Use:
-----------
1. Open index.html in a web browser
2. To add a new student:
   - Fill in all the required fields in the "Add New Student" form
   - Click "Add Student" button
3. To view students:
   - All students are displayed in the table below the form
4. To filter students:
   - Select year and/or branch from the dropdowns
   - Click "Search" button
   - Click "Reset" to clear filters
5. To edit or delete a student:
   - Use the buttons in the "Actions" column of the table

Portal Specifications:
----------------------
- Total Branches: 2 (CSE Engineer and Elex Engineer)
- Total Years: 3 (1st, 2nd, 3rd)

Student Distribution:
---------------------
1st Year: 40 students (CSE and Elex)
2nd Year: 33 students (10 Elex and 23 CSE)
3rd Year: 42 students (17 Elex and 25 CSE)
   - Note: Elex has no female students
   - CSE has 12 female students in 3rd year

Technical Details:
------------------
- Built with HTML, CSS, and JavaScript
- No server-side dependencies
- Data is stored in browser memory (will reset on page refresh)
- For production use, a backend database would be needed
